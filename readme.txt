== Description ==

Nemus Slider is a powerful, yet simple and easy-to-use WordPress slider plugin.

Great to create simple slideshows with photos, videos and HTML content. You can also create automated slides based on your WordPress posts.

Here is a video about the install process: http://www.youtube.com/watch?v=PVe5a5LHn4M

Main features

*   Beautiful user interface
*   Very simple to use
*   Instagram and Flickr integration
*   Easy to change the interface & button colours
*   Automated slides from posts and custom post types
*   Captions on images with multiple positions and animations
*   Responsive layout
*   Include within a WordPress theme
*   Youtube and Vimeo support
*   Localization ready
*   Filters & actions for developers
*   Horizontal and vertical sliders
*   Carousel mode
*   Autoplay function with custom interval
*   Automated plugin updates
*   Custom HTML slides with shortcode support
*   Advanced options

== Installation ==

This section describes how to install the plugin and get it working.

Installation

Extract the zipped package downloaded from WordPress.org to your desktop, in the extracted package you will find nemus-slider.zip file which is the WordPress plugin.

You can install the plugin in two ways:

FTP: Extract nemus-slider.zip file and upload the extracted folder to /wp-content/plugins/ folder on your server.
WordPress: Navigate to Plugins -> Add New-> Upload page. Select the nemus-slider.zip file. Press the Install Now button to upload and install the plugin.
After uploading the plugin, you have to activate it. Navigate to Plugins -> Installed Plugins, look for the Nemus Slider plugin and click Activate.
